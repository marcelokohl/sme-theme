1- Suba o config.json de /fontello para http://fontello.com/
2- Suba o novo ícone
3- Baixe o pacote
4- Suba a fonte em https://transfonter.org/
5- Converta para base64
6- Copie o src do css gerado pelo transfonter para o icons.scss
7- Copie as todas as classes dos icones do css do fontello, ex: icon-left-open:before para o icons.scss

Na dúvida, olhe o arquivo icons.scss q você irá entender o q precisa ser trocado.
